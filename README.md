# AI Company Policy Assistant

AI-powered document assistant built with ASP.NET Core, React, and Retrieval-Augmented Generation (RAG). Upload PDFs and ask questions in natural language. Uses vector embeddings and semantic search to retrieve relevant content and generate accurate, source-cited responses using an LLM.

Repository structure:

- frontend/                # React + TypeScript
- backend/                 # ASP.NET Core API
- docs/
  - architecture.md
  - api.md
  - screenshots/
- sample-documents/
- docker/
- .github/
  - workflows/
  - ISSUE_TEMPLATE/
- README.md
- docker-compose.yml
- LICENSE

See the docs/ directory for architecture and API details.
